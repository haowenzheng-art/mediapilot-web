
# MediaPilot - 快速启动指南（复制粘贴版）

## 最简单的方式：直接在 Anaconda Prompt 中复制粘贴命令

---

## 1. 中文版离线演示（推荐先试这个）

打开 Anaconda Prompt，复制粘贴：

```bash
cd c:\Users\19802\Desktop\ClaudeCodeTest\MediaPilot
python demo_cn_auto.py
```

---

## 2. 启动完整版后端

打开 Anaconda Prompt，复制粘贴：

```bash
cd c:\Users\19802\Desktop\ClaudeCodeTest\MediaPilot\backend
python main_full.py
```

启动后，浏览器访问：http://localhost:8000/docs

---

## 3. 启动简化版桌面端（需要先启动后端）

**新开一个** Anaconda Prompt 窗口，复制粘贴：

```bash
cd c:\Users\19802\Desktop\ClaudeCodeTest\MediaPilot
python desktop\gui_simple.py
```

---

## 4. 快速测试（验证环境）

```bash
cd c:\Users\19802\Desktop\ClaudeCodeTest\MediaPilot
python test_ascii.py
```

---

## 提示

- 如果后端未启动，桌面端会自动使用模拟数据
- 按 Ctrl+C 可以停止正在运行的程序
- 每个程序需要单独的 Anaconda Prompt 窗口
