
# MediaPilot 快速开始指南

## 🚀 三分钟快速启动

### 第一步：安装依赖

双击运行以下脚本（或手动执行）：

```bash
cd MediaPilot
pip install -r requirements.txt
```

### 第二步：启动后端服务

**方式A：使用脚本（推荐）**
```
双击：启动后端.bat
```

**方式B：手动启动**
```bash
cd backend
python main.py
```

后端服务将在 http://localhost:8000 启动

### 第三步：启动桌面端

**方式A：使用脚本（推荐）**
```
双击：启动桌面端.bat
```

**方式B：一键启动（后端+桌面端）**
```
双击：快速启动.bat
```

**方式C：手动启动**
```bash
cd desktop
python main.py
```

## 📋 使用说明

### 1. 配置AI服务

首次使用请在「设置」页面配置AI API：

- 点击左侧导航栏的「⚙️ 设置」
- 点击「🔑 配置API」
- 选择API类型并填写相关信息

**支持的AI服务：**
- Anthropic Claude
- OpenAI / GPT
- 火山方舟

### 2. 功能模块

| 模块 | 功能描述 |
|------|---------|
| 🔥 热点追踪 | 搜索行业最近热点话题 |
| 🎯 对标账号 | 搜索竞品账号，支持Excel导出 |
| 🎬 视频分析 | 分析爆款视频，生成逐字稿 |
| 🎤 音视频转写 | 上传mp3/mp4，一键转文字并生成大纲 |
| ✍️ 内容生成 | 输入选题，AI生成分镜头脚本和文案 |

### 3. API文档

后端API文档访问：http://localhost:8000/docs


## 🔧 常见问题

### Q: 提示「后端服务未启动」
A: 请先运行「启动后端.bat」，确保后端在 http://localhost:8000 正常运行

### Q: 没有API Key可以用吗？
A: 可以！部分基础功能使用模拟数据可以正常体验，内容生成需要配置AI服务

### Q: 支持哪些音视频格式？
A: mp3、mp4、wav、m4a、mov

### Q: Excel导出在哪里？
A: 在「对标账号」模块，搜索结果后点击「📥 导出Excel」
