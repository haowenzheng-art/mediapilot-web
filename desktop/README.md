
# MediaPilot 桌面端

## 快速开始

### 1. 安装依赖
```bash
pip install -r ../requirements.txt
```

### 2. 启动后端
先运行后端服务（见 backend/README.md）

### 3. 启动桌面端
```bash
python main.py
```

或使用Windows批处理脚本:
- `快速启动.bat` - 一键启动
- `启动桌面端.bat` - 仅启动桌面端

## 功能模块

- 📊 **仪表盘** - 功能概览
- 🔥 **热点追踪** - 搜索行业热点
- 🎯 **对标账号** - 搜索竞品账号，导出Excel
- 🎬 **视频分析** - 爆款视频分析
- 🎤 **音视频转写** - 语音转文字，生成大纲
- ✍️ **内容生成** - 生成分镜头脚本
- 📅 **内容日历** - 发布排期规划

## 配置说明

首次使用请在「设置」页面配置AI API，支持：
- Anthropic Claude
- OpenAI / GPT
- 火山方舟
